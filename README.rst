# ruckus-guest-py
Ruckus Guest Pass Generator

## Installation

.. code-block:: console

    $ pip install -U -r requirements.txt
    
## Config

Set "zd_username" and "zd_password" in ruckus.py

# Running

.. code-block:: console

    $ python ruckus.py
