# ruckus-guest-py
Ruckus Guest Pass Generator

## Requirements

```
python 2.7
```

## Installation

```
$ pip install -U -r requirements.txt
```
    
## Config

Set "zd_username", "zd_password" and "zd_url" in ruckus.py

# Running

```
$ python ruckus.py
```
